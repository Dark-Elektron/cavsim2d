Wakefield Analysis
==================

.. note::

   This section is a placeholder. Wakefield and impedance quantities are
   currently computed with the external time-domain solver
   `ABCI <https://abci.kek.jp/abci.htm>`_; its numerical formulation is not
   documented here yet. See :doc:`/wakefield` for the interface and the
   normalised result schema, and the eigenmode impedance reconstruction
   (:doc:`/eigenmode`) for the equivalent-circuit spectrum that can be compared
   against a wakefield solve.
